# Rock, Paper, Scissors

This is a solution to the Rock, Paper, Scissors Game from [Frontend Mentor](https://www.frontendmentor.io) challenges.

Try to win!

## Rules

If you win, you gain 1 point. If the computer wins, you lose one point.

## Original

- Paper beats Rock
- Rock beats Scissors
- Scissors beats Paper
